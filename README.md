### Space Image App ###

This is my final project for the React course (Spring 2024) at ReDI School of Digital Integration.
To see the result please visit: https://spaceimagesappredireactfinalproject.vercel.app/

# Overview

Space Image App allows users to search for images taken by the Hubble Space Telescope and like their favorite images. By default, it fetches data from the Unsplash API using the keywords ´nasa´ and ´hubble´. Users can sort results using filter categories or a search field. 

# Problem

The original idea was to create an app to compare images of the same space object taken by the James Webb Space Telescope and the Hubble Space Telescope. However, due to difficulties in obtaining useful data from the Nasa API, I modified the concept to apply the React concepts learned during the course, including hooks, routing, and state management.

# Setup

1. Clone the repository
2. Run "npm install"
3. Create a ".env.local" file with your Unsplash API key":
   "VITE_UNSPLASH_KEY=your_key_here"
4. Run "npm run dev"
